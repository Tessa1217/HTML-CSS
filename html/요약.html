HTML tag 요약
태그(tag) 설명 사용 예
html 웹 문서의 시작과 끝 (html lang="ko")...(/html)
head 문서 관련 정보 나열 (head)(meta charset="utf-8")(title)...(/title)(link)...(/head)
body 문서 몸통(화면 표시 내용) (body)...(/body)
header 문서 내용 머리 부분(로고, 메뉴) (header)(nav)메뉴(/nav)(/header)
section 본문 영역 (secion)...(article)내용(/article)...(/section)
footer 제작정보, 저작권 정보 (footer)...(/footer)
aside 본문 이외의 정보(광고, 링크) (aside)...(/aside)
hn 제목 태그 (h1)제목(/h1) 글자 크기: h1>h2>h3>h4>h5>h6-
p 문단 표시(앞뒤 빈 줄 삽입) (p)나는 어린 왕자 입니다(/p)
br 줄 바꿈 (p) 새롭게 ... (br)...(/p)
b, i 글자 굵게, 기울이기 (b)진하게(b), (i)기울이기(/i)
img 이미지 삽입 (img src = "강아지.png" width = "200" height = "100" alt = "강아지 그림" title = "튤팁")
a 하이퍼링크 설정 (a href = http://www.naver.com target = "blank" title = "튤팁")(/a)
pre 소스에 표시된 대로 표시 (pre)...(/pre)
mark 형광펜 효과 (mark)중요한 내용(/mark)
span 영역 묶기(스타일 적용) (span)어린 왕자(/span)
